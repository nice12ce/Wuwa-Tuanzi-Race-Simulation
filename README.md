## 🎮 核心规则速览
- **赛道**：1~32 格环形赛道，起点 1，终点 32。
- **角色**：6 名普通团子（千咲、莫宁、琳奈、爱弥斯、守岸人、珂莱塔） + 1 名捣乱 NPC（布大王）。
- **行动**：每人每回合投掷骰子（带技能修正），携带同格内高于自己的所有团子一起移动。
- **特殊格**：
  - 🟢 **绿格**（3,11,16,23）：全组前进 1 格  
  - 🔴 **红格**（10,28）：全组后退 1 格  
  - 🌀 **裂隙**（6,20）：随机打乱该格内堆叠顺序
- **冲线**：一组团子到达/越过 32 格时，**堆叠最高者获胜**，比赛立即结束。
- **布大王**：第 3 回合起从终点逆行，撞到团子会全部带走；若落后于所有团子则传送回起点。
